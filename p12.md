<h2>Dasar Teori</h2>
<p>Pada pertemuan ini, kita membangun sistem autentikasi sederhana (registrasi dan login) menggunakan <strong>Room Database</strong> untuk menyimpan data pengguna secara lokal. Fitur utama yang dipelajari adalah penggunaan <code>SharedFlow</code> di ViewModel untuk mengirimkan <em>one-shot events</em> seperti pesan Toast ke UI.</p>

<table>
  <tr><th>Konsep</th><th>Penjelasan</th></tr>
  <tr><td><code>SharedFlow</code></td><td>Hot stream yang dapat memancarkan nilai ke banyak kolektor; ideal untuk event sekali pakai seperti Toast, navigasi, atau dialog</td></tr>
  <tr><td><code>MutableSharedFlow</code></td><td>Versi SharedFlow yang dapat memancarkan nilai menggunakan fungsi <code>emit()</code></td></tr>
  <tr><td><code>LaunchedEffect</code></td><td>Composable side-effect yang menjalankan kode di coroutine saat composable pertama kali masuk komposisi atau saat key berubah</td></tr>
  <tr><td><code>collectLatest</code></td><td>Mengumpulkan Flow dan membatalkan pengumpulan sebelumnya ketika nilai baru tiba</td></tr>
  <tr><td>Password Hashing</td><td>Teknik menyimpan password dalam bentuk hash (bukan teks asli) untuk keamanan; menggunakan <code>MessageDigest</code></td></tr>
  <tr><td><code>@Query</code> dengan kondisi</td><td>Query Room dengan parameter untuk mencari pengguna berdasarkan email dan password hash</td></tr>
</table>

<div class="step"><strong>Langkah 4:</strong> Buat fungsi hash password sederhana menggunakan SHA-256.</div>

<code class="mono">object PasswordUtils {
    fun hashPassword(password: String): String {
        val digest = MessageDigest.getInstance("SHA-256")
        val hashBytes = digest.digest(password.toByteArray(Charsets.UTF_8))
        return hashBytes.joinToString("") { "%02x".format(it) }
    }

    fun isValidEmail(email: String): Boolean =
        android.util.Patterns.EMAIL_ADDRESS.matcher(email).matches()

    fun isValidPassword(password: String): Boolean = password.length >= 8
}</code>

<div class="step"><strong>Langkah 5:</strong> Buat <code>AuthViewModel</code> yang menggunakan <code>MutableSharedFlow</code> untuk mengirim event pesan ke UI.</div>

<code class="mono">sealed class AuthEvent {
    data class ShowToast(val message: String) : AuthEvent()
    object NavigateToHome : AuthEvent()
    object NavigateToLogin : AuthEvent()
}

class AuthViewModel(private val dao: UserDao) : ViewModel() {

    private val _event = MutableSharedFlow<AuthEvent>()
    val event: SharedFlow<AuthEvent> = _event.asSharedFlow()

    var isLoading by mutableStateOf(false)
        private set

    fun register(fullName: String, email: String, password: String, confirmPassword: String) {
        viewModelScope.launch {
            when {
                fullName.isBlank() -> emit(AuthEvent.ShowToast("Nama tidak boleh kosong"))
                !PasswordUtils.isValidEmail(email) -> emit(AuthEvent.ShowToast("Format email tidak valid"))
                !PasswordUtils.isValidPassword(password) -> emit(AuthEvent.ShowToast("Password minimal 8 karakter"))
                password != confirmPassword -> emit(AuthEvent.ShowToast("Konfirmasi password tidak cocok"))
                else -> {
                    isLoading = true
                    val isEmailTaken = dao.isEmailRegistered(email) > 0
                    if (isEmailTaken) {
                        emit(AuthEvent.ShowToast("Email sudah terdaftar"))
                    } else {
                        dao.insertUser(
                            User(
                                fullName = fullName.trim(),
                                email = email.trim().lowercase(),
                                passwordHash = PasswordUtils.hashPassword(password)
                            )
                        )
                        emit(AuthEvent.ShowToast("Registrasi berhasil! Silakan login."))
                        emit(AuthEvent.NavigateToLogin)
                    }
                    isLoading = false
                }
            }
        }
    }

    fun login(email: String, password: String) {
        viewModelScope.launch {
            when {
                email.isBlank() || password.isBlank() -> emit(AuthEvent.ShowToast("Email dan password wajib diisi"))
                else -> {
                    isLoading = true
                    val user = dao.findByEmailAndPassword(
                        email.trim().lowercase(),
                        PasswordUtils.hashPassword(password)
                    )
                    if (user != null) {
                        emit(AuthEvent.ShowToast("Selamat datang, ${user.fullName}!"))
                        emit(AuthEvent.NavigateToHome)
                    } else {
                        emit(AuthEvent.ShowToast("Email atau password salah"))
                    }
                    isLoading = false
                }
            }
        }
    }

    private suspend fun emit(event: AuthEvent) {
        _event.emit(event)
    }
}</code>

<div class="step"><strong>Langkah 6:</strong> Buat <code>LoginScreen</code> yang mengumpulkan <code>SharedFlow</code> event menggunakan <code>LaunchedEffect</code>.</div>

<code class="mono">@Composable
fun LoginScreen(
    viewModel: AuthViewModel,
    onNavigateToRegister: () -> Unit,
    onNavigateToHome: () -> Unit
) {
    val context = LocalContext.current
    var email by remember { mutableStateOf("") }
    var password by remember { mutableStateOf("") }
    var isPasswordVisible by remember { mutableStateOf(false) }

    // Mengumpulkan event dari SharedFlow — hanya dijalankan sekali saat composable masuk
    LaunchedEffect(Unit) {
        viewModel.event.collect { event ->
            when (event) {
                is AuthEvent.ShowToast ->
                    Toast.makeText(context, event.message, Toast.LENGTH_SHORT).show()
                AuthEvent.NavigateToHome -> onNavigateToHome()
                AuthEvent.NavigateToLogin -> { /* tidak relevan di layar ini */ }
            }
        }
    }

    Scaffold { innerPadding ->
        Column(
            modifier = Modifier
                .fillMaxSize()
                .padding(innerPadding)
                .padding(24.dp),
            horizontalAlignment = Alignment.CenterHorizontally,
            verticalArrangement = Arrangement.Center
        ) {
            Text("Login", fontSize = 28.sp, fontWeight = FontWeight.Bold)
            Spacer(modifier = Modifier.height(32.dp))
            OutlinedTextField(
                value = email,
                onValueChange = { email = it },
                label = { Text("Email") },
                keyboardOptions = KeyboardOptions(keyboardType = KeyboardType.Email),
                singleLine = true,
                modifier = Modifier.fillMaxWidth()
            )
            Spacer(modifier = Modifier.height(12.dp))
            OutlinedTextField(
                value = password,
                onValueChange = { password = it },
                label = { Text("Password") },
                visualTransformation = if (isPasswordVisible)
                    VisualTransformation.None
                else PasswordVisualTransformation(),
                trailingIcon = {
                    IconButton(onClick = { isPasswordVisible = !isPasswordVisible }) {
                        Icon(
                            if (isPasswordVisible) Icons.Default.Visibility
                            else Icons.Default.VisibilityOff,
                            contentDescription = null
                        )
                    }
                },
                singleLine = true,
                modifier = Modifier.fillMaxWidth()
            )
            Spacer(modifier = Modifier.height(24.dp))
            Button(
                onClick = { viewModel.login(email, password) },
                modifier = Modifier.fillMaxWidth().height(48.dp),
                enabled = !viewModel.isLoading
            ) {
                if (viewModel.isLoading) {
                    CircularProgressIndicator(modifier = Modifier.size(20.dp), color = Color.White)
                } else {
                    Text("Masuk")
                }
            }
            Spacer(modifier = Modifier.height(16.dp))
            TextButton(onClick = onNavigateToRegister) {
                Text("Belum punya akun? Daftar di sini")
            }
        }
    }
}</code>

<h2>Penjelasan Kode</h2>

<h3>SharedFlow untuk One-Shot Events</h3>
<p>Berbeda dengan <code>StateFlow</code> yang menyimpan nilai terakhir dan memutar ulangnya untuk kolektor baru, <code>SharedFlow</code> tidak menyimpan nilai (kecuali dikonfigurasi dengan <code>replay</code>). Ini sangat cocok untuk event sekali pakai seperti Toast — kita tidak ingin pesan "Login berhasil" muncul lagi ketika layar di-rekomposisi.</p>

<h3>LaunchedEffect untuk Mengumpulkan Event</h3>
<p><code>LaunchedEffect(Unit)</code> menjalankan block coroutine satu kali saat composable pertama kali masuk komposisi. Di dalam block tersebut, kita memanggil <code>viewModel.event.collect { ... }</code> yang berjalan terus-menerus sambil menunggu event baru. Ketika composable keluar dari komposisi, coroutine ini dibatalkan secara otomatis.</p>

<h3>Password Hashing</h3>
<p>Menyimpan password dalam bentuk teks biasa sangat berbahaya. Fungsi <code>PasswordUtils.hashPassword</code> menggunakan <code>MessageDigest.getInstance("SHA-256")</code> untuk mengubah password menjadi string heksadesimal 64 karakter. Hash bersifat satu arah — tidak dapat dikembalikan ke password asli. Saat login, password yang dimasukkan di-hash terlebih dahulu sebelum dibandingkan dengan nilai di database.</p>
